三、遇到的问题
初次执行 npm 命令提示无法识别，排查后确认是 Node.js 环境未配置系统环境变量，重装并勾选自动配置 PATH 后解决。
删除远程 master 分支被拒绝，原因是该分支当时为仓库默认分支，进入 GitHub 仓库设置修改默认分支为 main 后才可以正常删除。
GitHub Desktop 点击拉取代码弹出 Unable to merge unrelated histories 冲突，使用终端带参数拉取命令完成历史合并，两端代码同步。
